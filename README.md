# Katakana Dash

A fast-paced Japanese katakana learning game. One minute, sharp recall!

## Play Online

Visit the live app at: **https://ljnrg23-pixel.github.io/Katakana**

### On iPhone:
1. Open the link in Safari
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. Tap "Add"

The app will now appear on your home screen like a native app!

## Features

- **Two Game Modes**:
  - **Kana to Romaji**: See katakana, type the romaji
  - **Romaji to Kana**: See romaji, tap the correct katakana

- **60-Second Rounds**: Fast-paced gameplay with increasing difficulty
- **Score Tracking**: Your best scores are saved locally
- **Streak Counter**: Track your consecutive correct answers
- **Offline Support**: Works without internet using a service worker
- **Mobile Optimized**: Perfect for learning on your iPhone

## How to Use

1. **Choose a Game Mode** on the main menu
2. **Answer Questions** as quickly as possible (type your answer or tap the choice)
3. **Beat Your Best Score** across multiple rounds

## Scoring

- +1 point per correct answer
- Bonus points for streaks: +1 point for every 5 consecutive correct answers
- Your best score per mode is saved automatically

## Technical Details

- Built with vanilla JavaScript, HTML, and CSS
- Uses localStorage for score persistence
- Service Worker enables offline functionality
- PWA (Progressive Web App) support for home screen installation
